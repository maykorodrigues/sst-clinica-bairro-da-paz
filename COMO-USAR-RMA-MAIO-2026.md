# 📖 COMO USAR O RMA MAIO/2026

Relatório Mensal de Avaliação (RMA) — SST Clínica Bairro da Paz

---

## 🎯 3 Versões Disponíveis

### 1️⃣ **APRESENTAÇÃO INTERATIVA** (Recomendado)

**Arquivo:** `RMA-MAIO-2026.html`  
**Onde:** Abrir no navegador (Chrome, Edge, Firefox — sem internet necessária)

#### Como Usar:
```
1. Abra o arquivo HTML no navegador
2. Navegue com:
   → Setas do teclado (← →)
   → Botões "Anterior" / "Próximo" na tela
   → Navegação rápida: clique no slide counter
3. Slides com checklist são interativos — clique para marcar
4. Tempo de apresentação: 30–40 minutos
```

#### Estructura de Slides:
- **Slide 1:** Capa
- **Slides 2–3:** Resumo executivo + KPIs
- **Slides 4–7:** Análise (funil, receita, origem, retenção)
- **Slides 8–9:** Equipe + NPS
- **Slides 10–11:** Benchmarking + Riscos
- **Slides 12–14:** Insights + Plano de ação
- **Slides 15–17:** Decisões + Checklist + Apêndice

**Dicas:**
- Use modo tela cheia (F11) para melhor visualização
- Imprima como PDF se quiser compartilhar later
- Checklist é apenas visual — não salva automaticamente

---

### 2️⃣ **POWERPOINT** (Compatibilidade)

**Arquivo:** `RMA-MAIO-2026.pptx`  
**Quando Usar:** Compartilhamento, impressão, compatibilidade com Office

#### Como Usar:
```
1. Abra no PowerPoint / Google Slides / LibreOffice
2. Use apresentação modo full-screen (F5)
3. Navigate com setas ou cliques
4. Exporte como PDF se quiser distribuir por email
```

**Dicas:**
- Mais compacto (9 slides) que HTML (17 slides)
- Melhor para imprimir ou editar
- Mantém design e tabelas intactas

---

### 3️⃣ **MARKDOWN** (Referência Detalhada)

**Arquivo:** `RMA-MAIO-2026.md`  
**Quando Usar:** Leitura completa, busca rápida, citações

#### Como Usar:
```
1. Abra em editor Markdown (Obsidian, VSCode, etc.)
2. Navegue usando índice automático
3. Use Ctrl+F para buscar palavras-chave
4. Copie tabelas/seções conforme necessário
```

**Seções:**
- Resumo Executivo (início)
- KPIs + Funil + Receita
- Análise por área (Equipe, Retenção, Churn)
- Riscos + Bloqueadores
- Plano de Ação
- Decisões Solicitadas
- Apêndice com dados brutos

## ⚡ USO RÁPIDO (5 MINUTOS)

Se você tem apenas 5 minutos, leia nesta ordem:

1. **`RMA-MAIO-2026-RESUMO-EXECUTIVO.md`** (1 página)
   - Números-chave
   - O que está funcionando/não
   - 3 ações críticas
   - Gate decision

2. **Slide 2 do HTML** (Resumo Executivo com KPIs)
   - Visualização gráfica
   - Status por métrica

3. **Slides 15–16 do HTML** (Decisões)
   - Marque sua resposta no checklist
   - Pronto para decidir

---

## 🎤 PARA APRESENTAR AO ROGÉRIO (30–40 min)

### Fluxo Recomendado

**[0–5 min]** Capa + Resumo Executivo (Slides 1–2)  
→ Estabeleça tom: "Semana 1 positiva, mas volume ainda baixo"

**[5–15 min]** KPIs + Funil (Slides 3–4)  
→ Destaque: taxa conversão 21% acima meta

**[15–25 min]** Riscos + Bloqueadores + Plano de Ação (Slides 8–12, 14)  
→ Mostre que você tem controle da situação

**[25–35 min]** Decisões Solicitadas (Slide 15)  
→ Peça aprovações: Coordenador? Budget? Tenex?

**[35–40 min]** Checklist + Próximas Ações (Slide 16)  
→ Feche com clareza: GO / CONTINUE / NO-GO?

### Dicas de Apresentação

- **Comece com número-chave (3 cartões, 21% conversão)**
  - Deixa claro que houve movimento real, não teoria
  
- **Mostre a progressão de leads** (Funil Slide 4)
  - Demonstra que não há gargalo morto
  
- **Destacue riscos E mitigações** (Slide 12)
  - Mostra controle + planejamento
  
- **Termine com checklist interativo** (Slide 16)
  - Engaja Rogério na decisão final

- **Deixe PowerPoint/PDF como backup**
  - HTML pode ter problema em some projectors
  - Sempre tenha versão impressa também

## 💾 EXPORTAR / COMPARTILHAR

### Para Enviar por Email

**Opção 1: PDF do HTML**
```
1. Abra RMA-MAIO-2026.html no navegador
2. Ctrl+P → Salvar como PDF
3. Envie o PDF
```

**Opção 2: PowerPoint Nativo**
```
1. Abra RMA-MAIO-2026.pptx
2. File → Export / Share
3. Envie o arquivo .pptx ou PDF
```

### Para Imprimir

**Recomendado:**
1. Exporte HTML como PDF (Ctrl+P)
2. Imprima em cores (preferir) ou preto-branco
3. 17 páginas (se HTML) ou 9 (se PowerPoint)
4. **Dica:** Imprima apenas Slides 1–2, 4, 8–9, 15–16 se papel for limitado

## ⚙️ PERSONALIZAR DADOS

Se quiser atualizar dados após 12/05:

### No HTML (RMA-MAIO-2026.html)
- Abra em editor de texto (VSCode)
- Busque por `<div class="value">` para números
- Exemplo: `<div class="value">3</div>` → mude para `4`
- Salve e recarregue no navegador

### No PowerPoint
- Abra normalmente
- Clique em textbox com número
- Edite direto
- Salve

### No Markdown
- Abra em editor de texto
- Edite tabelas/números
- Salve

---

## 📞 SUPORTE

Se HTML não abrir no navegador:
1. Tente outro navegador (Chrome, Edge)
2. Use a versão PowerPoint como fallback
3. Revise markdown como referência

Se tabelas saírem errado ao imprimir:
1. Use PDF em vez de impressão direta
2. Ajuste orientação (paisagem vs retrato)
3. Reduza zoom no navegador antes de imprimir (Ctrl+-)
