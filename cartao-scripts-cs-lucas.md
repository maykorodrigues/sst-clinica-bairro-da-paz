## COMO USAR ESTE CARTÃO

1. **Copiar o script que faz sentido** (ex: cliente novo = Script 1)
2. **Preencher [ASSIM]** com dados reais (nome, CPF, data, etc.)
3. **Colar direto no WhatsApp** — não precisa digitar tudo
4. **Personalizar 1 frase** — adicionar algo seu torna genuíno
5. **Enviar** — e registrar na tabela de CS (Notion)
