## 📊 Os 3 Planos Propostos

### **PLANO 1: INDIVIDUAL** (Âncora de entrada)

| Item | Valor |
|---|---|
| **Nome** | SST Card Individual |
| **Preço mensal** | **R$ 34,90** |
| **Beneficiários** | 1 pessoa (titular) |
| **Acesso** | Clínica + Telemedicina |
| **Caso de uso** | Pessoa que quer começar sozinha; testa benefício |
| **Conversão esperada** | 60%–70% dos leads |

**Diferenciais:**
- Acesso completo a clínico, telemedicina, pediatra (conforme disponível)
- Cria hábito de uso baixo-risco
- Gateway para upgrade posterior

---

### **PLANO 2: FAMILIAR 4–5 PESSOAS** (NOVO — Teste estratégico)

| Item | Valor |
|---|---|
| **Nome** | SST Card Família Pequena |
| **Preço mensal** | **R$ 65,00** (valor proposto — validar com Karine) |
| **Beneficiários** | 4–5 pessoas (titular + dependentes) |
| **Acesso** | Clínica + Telemedicina + cobertura expandida |
| **Caso de uso** | Casal + 2–3 filhos OU mãe + filhos |
| **Conversão esperada** | 20%–25% dos leads |

**Estrutura operacional:**
- Titular tem acesso completo + controle de dependentes
- Dependentes usam sem autorização para cada consulta (modelo de cartão)
- Relatório de uso por pessoa
- SMS/WhatsApp alertando quando alguém da família usa

**Por que testar agora no Bairro da Paz:**
1. Validar demanda (família de 4–5 pessoas é padrão na região)
2. Aumentar ticket médio (R$34,90 → R$65,00 = +86% receita por titular)
3. Maior stickiness (família tem mais razão para não cancelar)
4. Gateway para Plano 3

## 🎯 Estratégia de Rollout

### **Fase 1: Bairro da Paz (Agora — Maio–Junho)**
- Lançar os 3 planos simultaneamente
- Mix esperado: 65% Individual | 20% Família 4–5 | 15% Família 10
- Meta: Validar operação, prova de conceito
- KPI: 20+ adesões com mix equilibrado

### **Fase 2: Monsílio (Julho–Agosto) — NÃO MEXER AGORA**
- Mantém estrutura atual (2 planos)
- Depois que Bairro da Paz der certo, volta para otimizar
- **Decisão:** "Se Monsílio está funcionando lá, deixa como está. Depois que a gente dá certo aqui, a gente volta para lá."

### **Fase 3: Expansão Interior/Santa Amara (Setembro+)**
- Após case de sucesso do Bairro da Paz
- Replicar estrutura 3 planos em novos bairros
- Potencial: 3–4 novas filiais com essa estrutura

### **Fase 4: Investor Gilberto (Outubro+)**
- Entra APENAS após case de sucesso consolidado
- Traz capital para acelerar expansão
- Estrutura financeira já validada = proposta mais forte
