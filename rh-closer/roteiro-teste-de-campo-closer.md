# Roteiro Teste de Campo — Closer SST Card

> **Última etapa antes do GO.** A entrevista comportamental aprovou cultura; a simulação mostra técnica. **O teste de campo mostra a verdade**: quem realmente aborda estranho na rua, contorna objeção real e **pede o fechamento** — com cliente de verdade, no Bairro da Paz.
>
> **Aplicar a TODAS as candidatas com a MESMA régua** (Safira, Vanesca + eventual 3ª) para comparação lado a lado. Mesmo território, mesmo material, mesmo tempo, mesma meta.
>
> **Conduz:** Rogério (presencial, decisão) + Mayko (alinhamento técnico/remoto) · **Duração:** ~meio período por candidata (3–4h) · **Local:** Bairro da Paz (rua + comércio + porta-a-porta)

---

## Resumo executivo — o que o campo decide

A rua revela 4 coisas que nenhuma entrevista mostra:

1. **Coragem de abordar a frio** — para na frente de um estranho e inicia a conversa, ou trava?
2. **Pede o fechamento** — leva a pessoa à adesão/agendamento, ou só "informa e sai"?
3. **Resiliência ao "não"** — depois de 5–10 nãos seguidos, mantém energia ou murcha?
4. **Números reais** — quantas abordagens viram conversa, quantas conversas viram adesão.

> ⚠️ **Pontos de atenção já mapeados (corrigir/observar no campo):**
> - **Vanesca (júnior):** na simulação **não pediu o fechamento** e **confundiu a tabela de preços**. No campo: ela decorou os preços? Ela pede o sim? — ver [[scoring-vanesca-15-06-2026]].
> - **Safira (sênior):** declarou 10+ anos de consignado mas **sem números**. No campo: ela realmente caça (porta fria) ou só faz receptivo? — ver [[scoring-safira-11-06-2026]].

## Regras de justiça (para a comparação valer)

| Regra | Por quê |
|---|---|
| **Mesma área e horário** para todas (ex.: comércio da via principal + 2 ruas residenciais, manhã 9h–12h) | Tira a sorte do território da equação |
| **Mesmo material** (cartão de bolso, tabela de preços, folheto) entregue igual | Todas partem do mesmo ponto |
| **Mesmo mini-treino de 30 min** antes de sair | Nivela a júnior sem entregar a venda pronta |
| **Mesma meta declarada** ("o máximo de adesões/agendamentos em 3h") | Mede ambição + execução |
| **Mesmo observador** (Rogério acompanha cada uma o mesmo tempo) | Régua única |
| **Mesma ficha de scoring** preenchida ao fim de cada uma | Comparação objetiva |
| Rodar candidatas **em dias/turnos separados** (não juntas) | Evita constrangimento e cópia |

> 💡 **Ajuda de custo:** combinar com Rogério um valor simbólico de diária (ex.: R$50) pelo dia de teste, independente do resultado. Mostra respeito e atrai gente boa. **Adesões fechadas no teste contam como venda real** (e já geram comissão se contratada) — deixar isso claro motiva.

---

## Estrutura do dia (por candidata)

| Bloco | Tempo | Objetivo |
|---|---|---|
| 1. Briefing + entrega de material | 15 min | Contexto, regras, meta do dia |
| 2. Mini-treino do pitch (30-30-30) | 30 min | Nivelar pitch, preços e fechamento |
| 3. Aquecimento observado (Rogério faz 1, ela faz 1) | 20 min | Modelar + soltar |
| 4. Campo solo observado | 3h | ⭐ O teste real — abordagens de verdade |
| 5. Debrief + autocrítica | 15 min | Como ela se avalia |
| 6. Scoring comparativo (sem a candidata) | 10 min | Preencher ficha + ranking |

---

## BLOCO 1 — Briefing + material (15 min)

Falar para a candidata (mesmo script para todas):
> "Hoje é prática de verdade. Você vai sair comigo aqui no Bairro da Paz e oferecer o SST Card pra pessoas reais — comércio e moradores. Não tem problema errar, tem problema não tentar. A meta é simples: o **máximo de adesões ou agendamentos** que você conseguir em 3 horas. Eu fico por perto observando, mas quem fala é você."

**Entregar (igual para todas):**
- [ ] Cartão de bolso com pitch + preços (R$39,90 ind. / R$64,90 fam. + adesão R$35)
- [ ] Folheto/folder do SST Card
- [ ] Ficha de captação (nome, WhatsApp, plano de interesse) — para registrar leads/adesões
- [ ] Crachá/identificação da SST (legitimidade na porta)

---

## BLOCO 2 — Mini-treino do pitch (30 min) — formato 30-30-30

> Nivela a júnior **sem entregar a venda mastigada**. Todas recebem o mesmo treino.

1. **10 min — Produto e preços decorados.** Repetir até acertar de olhos fechados: Individual **R$39,90**, Família **R$64,90**, Adesão **R$35**. Benefícios: telemedicina + auxílio funeral + assistência veterinária + medicina com custo reduzido. Âncoras: *"R$1,33 por dia"* · *"R$25 a mais e todo mundo tem todos os benefícios"*.
   - 🔴 **Checar Vanesca aqui:** ela errou os preços na simulação. Só sai a campo quando acertar a tabela 3× seguidas.
2. **10 min — As 3 objeções e as respostas:** "tô apertada" / "e se eu pagar e não usar" / "vou no SUS de graça". Treinar a resposta de cada uma.
3. **10 min — O FECHAMENTO.** Treinar a frase de fechamento explícita: *"A senhora quer começar hoje no individual ou já garante a família toda?"* — **sem aceitar "vou pensar" sem reagir**.
   - 🔴 **Foco de todas, especialmente Vanesca:** a venda só termina quando pede o sim.

---

## BLOCO 3 — Aquecimento observado (20 min)

- **Rogério (ou Mayko) faz 1 abordagem modelo** com a candidata assistindo (10 min).
- **A candidata faz 1 abordagem** ao lado do Rogério, sem pressão de resultado, só pra soltar (10 min).

Observar já aqui: ela trava? Copia o modelo? Adapta com jeito próprio?

---

## BLOCO 4 — Campo solo observado (3h) ⭐ O TESTE REAL

A candidata aborda sozinha; Rogério acompanha a uma distância que permita ouvir sem interferir. **Não corrigir no meio** — anotar e devolver no debrief.

### Placar objetivo (contar ao vivo)

| Métrica | Contagem | Observação |
|---|---|---|
| **Abordagens iniciadas** (parou e falou) | | Volume bruto = coragem |
| **Conversas qualificadas** (ouviu o pitch até o fim) | | Engajou de verdade |
| **Objeções contornadas** (rebateu e seguiu) | | Não desistiu no 1º não |
| **Pedidos de fechamento** (pediu o sim explicitamente) | | ⭐ O sinal de closer |
| **Adesões fechadas** (assinou/pagou adesão) | | Resultado final |
| **Agendamentos** (marcou volta/visita à clínica) | | Resultado parcial válido |
| **Leads captados** (WhatsApp na ficha) | | Pipeline gerado |

### Régua comportamental (marcar ✅/⚠️/❌ ao longo das 3h)

| Comportamento | Avaliação |
|---|---|
| Aborda estranho sem travar | ☐✅ ☐⚠️ ☐❌ |
| Sorri / cria rapport rápido | ☐✅ ☐⚠️ ☐❌ |
| Identifica a necessidade antes de empurrar produto | ☐✅ ☐⚠️ ☐❌ |
| Fala os preços com segurança (sem gaguejar/errar) | ☐✅ ☐⚠️ ☐❌ |
| Contorna objeção com empatia (não discute) | ☐✅ ☐⚠️ ☐❌ |
| **PEDE o fechamento** (não só informa) | ☐✅ ☐⚠️ ☐❌ |
| Tenta a família (R$64,90), não só o individual | ☐✅ ☐⚠️ ☐❌ |
| Mantém energia após sequência de "nãos" | ☐✅ ☐⚠️ ☐❌ |
| Registra o lead/contato de quem não fechou na hora | ☐✅ ☐⚠️ ☐❌ |
| Conhece o bairro / cria conexão local | ☐✅ ☐⚠️ ☐❌ (vantagem Vanesca = nativa) |

> **🔴 KILLER do campo:** terminou 3h **sem pedir um único fechamento** = não é closer, por melhor que converse. Volume de abordagem zero (ficou com medo de abordar) = NO-GO.

---

## BLOCO 5 — Debrief + autocrítica (15 min)

Perguntar (mede consciência e treinabilidade — chave para a júnior):
1. "De 0 a 10, como você se avalia hoje? Por quê?"
2. "Qual foi a abordagem que você mais gostou? E a que travou?"
3. "Se voltasse agora pra rua, o que faria diferente?"
4. "Como você se sentiu nos 'nãos'?"
5. "Faria isso todo dia? Por quê?"

🟢 **Verde:** autocrítica honesta + vontade de voltar amanhã + identifica o próprio erro.
🚩 **Vermelho:** culpa o cliente/bairro · não quer repetir · não vê o que melhorar.

## BLOCO 6 — Scoring comparativo (sem a candidata — 10 min)

Preencher **a mesma ficha para cada candidata** logo após o campo:

| Dimensão | Peso | Score (0–10) | Pontos |
|---|---|---|---|
| Coragem de abordagem (volume + iniciativa) | x3 | __ | __ |
| **Fechamento (pediu o sim + converteu)** | x4 | __ | __ |
| Contorno de objeção real | x3 | __ | __ |
| Resiliência ao "não" (energia ao longo das 3h) | x2 | __ | __ |
| Rapport / conexão local | x2 | __ | __ |
| Autocrítica + treinabilidade (debrief) | x2 | __ | __ |
| **TOTAL** | | | **__/160 → __/100** |

> Converter: total ÷ 1,6 = score sobre 100.

### Tabela comparativa final (preencher com todas)

| Candidata | Abordagens | Conversas | Adesões | Agendamentos | Pediu fechamento? | Score campo /100 | Perfil |
|---|---|---|---|---|---|---|---|
| **Safira** (sênior) | | | | | | | Hunter — validar se caça mesmo |
| **Vanesca** (júnior) | | | | | | | Treinável — validar fechamento |
| *(3ª, se houver)* | | | | | | | |

### Decisão

| Cenário | Ação |
|---|---|
| **1 candidata se destaca claramente** (mais adesões + pediu fechamento) | GO para ela — proposta na semana |
| **Sênior converte + júnior mostra mais treinabilidade/fit local** | Considerar **júnior como closer + sênior como gestora**, ou contratar as duas em níveis diferentes (decisão Rogério) |
| **Empate técnico** | Validar referências + 2º mini-teste curto |
| **Nenhuma pede fechamento / converte** | NO-GO geral — reabrir pipeline (`divulgacao-vaga-closer.html`) |

> 💡 **Leitura estratégica:** o teste de campo pode não ser "ou uma ou outra". A operação Bairro da Paz pode comportar **Sênior (Safira) como gestora/closer âncora + Júnior (Vanesca) como closer em formação** — desde que o orçamento e o volume de leads justifiquem. Levar os dois cenários ao Rogério com os números na mão.
