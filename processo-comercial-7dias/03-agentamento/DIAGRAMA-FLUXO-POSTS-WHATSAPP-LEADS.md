## 🔗 CONEXÕES CRÍTICAS

### Conexão 1: Meta Suite → WhatsApp

**O que conecta:**
```
CTA do post: "Envia ZAP pra gente → [LINK]"

Link format:
https://wa.me/5571XXXXXXXXX?text=SST%20Card

Quando clica:
✅ Abre WhatsApp automaticamente
✅ Chat com número conectado ao GPTMaker
✅ Texto pré-digitado: "SST Card"
```

**Problema comum:**
- Link errado → não abre WhatsApp
- Número errado → abre para pessoa errada

**Solução:**
- Copiar link exato do GPTMaker
- Testar clicando antes de agendar

---

### Conexão 2: GPTMaker → Raquel

**O que conecta:**
```
GPTMaker notifica Raquel:
✅ Ding! "Novo lead: João Silva"

Raquel pode:
✅ Responder direto no chat
✅ Marcar como resolvido
✅ Encaminhar para Karine

Karine recebe no WhatsApp:
"João Silva quer saber mais sobre SST Card"
```

**Problema comum:**
- Raquel não vê mensagens (GPTMaker não tem notificação)
- Mensagem fica sem resposta > 5 min

**Solução:**
- Deixar GPTMaker aberto na semana
- Ativar notificações do navegador

---

### Conexão 3: Karine → Blingo

**O que conecta:**
```
Quando Karine fecha venda:
✅ Registra no Blingo (CRM)
✅ Anota origem: "Post segunda - Dor/Fila"

Raquel depois consulta:
"Quantos leads do post de segunda viraram clientes?"
"Qual post teve melhor conversão?"
```

**Problema comum:**
- Karine não anota origem
- Raquel não consegue rastrear

**Solução:**
- Treinar Karine em 5 min (só anotação)
- Usar campo "Nota" no Blingo
