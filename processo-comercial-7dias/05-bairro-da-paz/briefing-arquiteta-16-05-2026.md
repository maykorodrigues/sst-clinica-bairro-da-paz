# Briefing para Arquiteta — 2ª Filial SST Clínica Bairro da Paz

**Visita:** Sexta-feira, 16/05/2026 | **Contato:** Rogério Ferreira

> Este briefing deve ser entregue à arquiteta antes ou durante a visita para orientar o levantamento técnico e a proposta de layout.

---

## Resumo do Projeto

A SST Clínica está abrindo sua 2ª filial no **Bairro da Paz, Salvador/BA** — bairro de ~21.000 habitantes, trabalhadores informais, alta dependência do SUS. O modelo de negócio combina:

- **Clínica popular** (consultas + exames a preço acessível)
- **SST Card** (cartão de saúde — assinatura R$34,90 a R$199,90/mês)
- **Sala de telemedicina** (pronto atendimento remoto + consultas médicas presenciais)

A operação começa enxuta: atender e vender cartões desde o dia 1, com expansão gradual.

---

## Conceito de Layout — Decisão de Rogério (13/05)

> *"Minha ideia é já fazer uma sala para pronto atendimento de telemedicina e atendimento de médico. Enquanto isso, o processo vai rodando, a gente vai já atendendo e já vai vendendo os cartões."*

### Distribuição proposta

```
TÉRREO
├── Recepção / Cadastro SST Card (área de entrada)
├── Sala de Espera (mínimo 8–10 lugares)
├── Sala de Telemedicina + Atendimento Médico (1 sala polivalente)
└── Banheiro (exigência sanitária)

ANDAR SUPERIOR (se houver)
├── Escritório SST Card (operação Karine/equipe)
├── Sala de Vendas / Reunião com Embaixadores
└── Almoxarifado / Estoque de material
```

## Requisitos por Ambiente

### 1. Recepção / Cadastro SST Card

| Item | Especificação |
|---|---|
| Balcão de atendimento | 1 posto (1 atendente) com espaço para computador |
| Área para fila | Comportar 4–6 pessoas em pé |
| Display do cartão | Painel ou suporte para expositor SST Card |
| Sinalização | Local visível para faixa de preços e serviços |
| Ponto de internet | Cabo + tomadas para notebook e maquininha |

### 2. Sala de Espera

| Item | Especificação |
|---|---|
| Cadeiras | Mínimo 8 lugares (plástico resistente) |
| TV | Suporte para TV 32"+ (conteúdo de espera + divulgação SST Card) |
| Climatização | Ventilador de teto ou split (clima quente de Salvador) |
| Iluminação | Natural preferível + artifical fria |

### 3. Sala de Telemedicina + Atendimento Médico ⭐

Esta é a sala estratégica — precisa funcionar para dois fins:

**Modo A — Telemedicina (pronto atendimento remoto)**
- Monitor/TV 40"+ para videochamada com médico remoto
- Câmera de boa qualidade (capturar o paciente)
- Internet estável (fibra, ponto de cabo garantido)
- Espaço para o paciente sentar em frente à tela
- Maca simples para ausculta local

**Modo B — Consulta Médica Presencial**
- Espaço para médico + paciente (mínimo 9m²)
- Mesa/escrivaninha para o médico
- Maca de exame
- Lavatório ou área de higienização
- Privacidade total (porta com trava, sem vidro)

| Requisito Técnico | Detalhe |
|---|---|
| Área mínima | 12m² (polivalente) |
| Tomadas | Mínimo 4 (monitor, computador, equipamentos) |
| Internet | Ponto de cabo obrigatório (Wi-Fi como backup) |
| Climatização | Split obrigatório (conforto do paciente) |
| Iluminação | Fria + dimmer (exame clínico) |
| Privacidade | Porta sólida, sem janela para área de espera |

### 4. Escritório SST Card (andar superior ou ambiente separado)

| Item | Especificação |
|---|---|
| Mesas | 2–3 postos de trabalho (Karine + vendedor + estagiário) |
| Armário | Guardar contratos, documentação |
| Internet | Obrigatório |
| Privacidade | Separado do fluxo de pacientes |

## Exigências Regulatórias — Atenção

> A arquiteta deve considerar as normas da **ANVISA RDC 50/2002** para estabelecimentos de saúde.

| Item obrigatório | Observação |
|---|---|
| Banheiro exclusivo para pacientes | Separado do staff |
| Piso lavável em áreas clínicas | Consultório + sala de espera |
| Lavatório com torneira sem contato manual | Consultório médico |
| Ventilação adequada | Norma sanitária Salvador |
| Sinalização de emergência | Saída, extintor |
| Alvará sanitário (VISA Salvador) | Processo iniciado antes da reforma |
