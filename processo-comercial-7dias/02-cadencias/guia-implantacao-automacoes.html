🆘 Problemas Comuns e Soluções
O que fazer quando algo não funciona
▼
Webhook do Chatwoot não dispara o A1
Verificar: Chatwoot → Settings → Integrations → Webhooks → URL está ativa? Evento "Message Created" marcado? Clicar em "Test" para forçar um evento.
Notion retorna 404 no A1
Checar: (1) NOTION_PARCIAIS_DB_ID ou NOTION_CLINICA_DB_ID correto? (2) Database compartilhado com a integração "n8n SST Parser"? Verificar em DB → ··· → Connections.
Claude retorna erro de API Key no A1
Verificar ANTHROPIC_API_KEY no n8n. Testar diretamente em console.anthropic.com. Checar se a key não expirou.
A2 não encontra parciais no Notion
O filtro de data usa o formato YYYY-MM-DD com timezone São Paulo. Confirmar que as entradas do Notion têm a propriedade "Data" preenchida (não apenas o título). Executar A1 manualmente primeiro para gerar dados do dia.
Evolution API retorna erro 401 no A2/A3/A6/A7/A9
Checar EVOLUTION_API_KEY e EVOLUTION_INSTANCE. O nome da instância é case-sensitive. Confirmar que a instância está conectada (status = open) no painel Evolution.
Google Sheets retorna 403 no A3/A6/A7
A credencial Google Sheets no n8n não tem permissão. Reconectar: n8n → Credentials → Google Sheets → Reconnect. Se usar Service Account, verificar que ela tem acesso Editor à planilha.
A3 não atualiza a fase no Sheets
O nó de atualização busca a linha pelo campo whatsapp. Confirmar que o número está exatamente no formato 5571XXXXXXXXX (sem +, espaço, traço ou parênteses).
Typebot (A5/A8) não envia o webhook para o n8n
Confirmar que a URL no nó "Webhook" do Typebot está correta e acessível. Testar abrindo a URL no navegador (deve retornar erro 405 ou 200, não "não encontrado"). Verificar se o workflow n8n está ativo.
A9 não cria contato no Chatwoot
Verificar: (1) CHATWOOT_API_KEY é um token de admin? (2) CHATWOOT_ACCOUNT_ID = 1? (3) O número WhatsApp do lead tem pelo menos 10 dígitos? Ver o nó "Criar Contato" no log de execução do n8n.
A6 reenvia D+0 para membros que já receberam
A proteção é a coluna onboarding_iniciado = TRUE. Se estiver vazia por falha de atualização, preencher manualmente. O workflow ignora membros com esse campo = TRUE.
