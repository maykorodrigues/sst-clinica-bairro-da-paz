## 📋 COMO USAR

### Para a equipe (Raquel, Karine, Lucas):

1. **12h00** — Copiar template PARCIAL, preencher 3 primeiros dados e enviar no WhatsApp grupo MADIP
2. **Saída (17h00-17h30)** — Copiar template FINAL, preencher completo e enviar

### Para Mayko (Consolidação):

1. Receber 6 mensagens (parcial 12h + final 17h de cada um)
2. Compilar no template CONSOLIDADO
3. Arquivar em `historico-resultados-diarios/` com nome `resultado-DDMMYYYY.md`
4. Usar para relatório semanal de Rogério

## 💡 Dicas Rápidas

- ⏱️ **Tempo máximo:** 2 minutos para preencher
- 📱 **Formato:** Copiar/colar direto no WhatsApp (manter markdown simples)
- 🎯 **Honestidade:** Não inflar números — serve para ajustar processo, não punir
- 🚫 **Bloqueador:** Ser específico (ex: "Sem acesso ao Boom" vs "não consigo ativar")
- 📊 **Padrão:** Mesma estrutura todo dia = comparação fácil
